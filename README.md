# Rudy's 360 — Warmer Labels

A browser-based label generator for Rudy's 360 heated food warmers. Edit product names, prices, and categories, then print directly from Chrome.

**Live tool:** [ethansnider111.github.io/warmer-labels/Taco_Warmer_Labels.html](https://ethansnider111.github.io/warmer-labels/Taco_Warmer_Labels.html)

---

## How to use

1. Open the link above in Chrome
2. Edit label names, prices, and types using the input fields
3. Use the **Layout** button to toggle between eyebrow (category on top) and stacked (category below name)
4. Click **Print Labels** to print — make sure **Background graphics** is checked in Chrome's print dialog
5. All edits save automatically and persist between sessions

---

## Label fields

Each label has three fields:

- **Name** — the product (e.g. `BRISKET & EGG`)
- **Price** — displayed in the black bar at the bottom (e.g. `$2.99`)
- **Type** — category shown in red (e.g. `BREAKFAST TACO`). Choose from the dropdown or select **Custom...** to type your own. Leave blank for no category label.

---

## Managing shelves and labels

- **+ Add Label** — adds a new label slot to a shelf
- **+ Add Shelf** — adds a new shelf section (e.g. for a second warmer)
- **× button** — removes an individual label
- **Remove Shelf** — deletes an entire shelf section
- Shelf names are editable by clicking the yellow text at the top of each shelf block
- The header text (store name and title) is also editable by clicking it

---

## Printing tips

- Print on **cardstock** then **laminate** for durability in a warm environment
- Use **adhesive magnet strips** on the back to mount on the stainless steel warmer front
- Print orientation: **Landscape**
- Card size targets approximately **0.9"** tall to fit the warmer rail

---

## Updating the live tool

1. Make edits and download the updated `Taco_Warmer_Labels.html`
2. Go to the [warmer-labels repo](https://github.com/ethansnider111/warmer-labels) on GitHub
3. Click the file → **Edit** (pencil icon) or drag-and-drop a new version via **Add file → Upload files**
4. Commit the change — live site updates within a minute
